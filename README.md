# Linux System Monitoring & Automation

This project is a Linux-based system monitoring tool built using Bash scripting.
It automatically collects system information and logs it using cron jobs.

## Features
- Automated system health monitoring
- Disk usage tracking
- Memory usage tracking
- Process monitoring
- Cron-based automation
- Log generation

## Tools Used
- Linux
- Bash
- Cron
- Git & GitHub

## How it works
The script runs every 10 minutes using cron and appends system health data into a log file.

## Use Case
This project simulates real-world server monitoring used by DevOps engineers.
